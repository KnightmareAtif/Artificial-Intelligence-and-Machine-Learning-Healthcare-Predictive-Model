# Artificial-Intelligence-and-Machine-Learning-Healthcare-Predictive-Model
AIML-HealthGuard is an integrated diagnostic platform providing risk assessments for heart disease, lung conditions, and breast tumors. This notebook features high-performance models like XGBoost and ResNet50, integrated with SHAP for Explainable AI (XAI) to ensure all medical predictions are transparent and clinically actionable.

## Directory Descriptions
 - ### Brain Model
Contains the trained machine learning models and weight files specifically developed for brain-related diagnostics and analysis.

 - ### Brain_APP
The backend application files and API logic (typically app.py) required to serve the brain predictive models.

 - ### Dataset for Brain
Stores the raw and processed data used to train and validate the brain-related machine learning algorithms.

 - ### Dataset for Heart
Contains the datasets, including medical records or sensor data, used for heart disease prediction and analysis.

 - ### Dataset for Lungs
Housing for respiratory-related data, such as X-ray image classes or clinical data used for lung condition classification.

 - ### Heart Model
Includes the specific algorithms and saved model states utilized for heart health predictions and risk assessment.

 - ### Heart_APP
The backend service and integration scripts designed to process inputs and return results for the heart prediction module.

 - ### Lung Model
Contains the notebooks, source code, and visualization exports used to develop and evaluate the lung predictive models.

 - ### Lung_APP
The application logic and error-handling scripts (such as indices error management) for the lung diagnostic interface.

 - ### frontend
The user interface (UI) source code, including web components and styling, that allows users to interact with the predictive models.
